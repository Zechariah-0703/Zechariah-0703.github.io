1. 编程方面

1）Python

  • 高中生水平，能自己拼凑出解决中国剩余定理三素数RSA的代码

  • 昨天调试python程序时Vscode时常崩溃，发现是JAVA的环境变量PATH里的‘；’导致python的路径里出现了‘ ” ’，致使Python的库不能正常展开。重写了环境变量，原来的一句分成两句写，避免出现‘；’，解决问题。（所以为什么java的环境变量会影响到python啊？？？神奇）

  • 前几天python的扩展自动更新了，Vscode又崩溃，安装老版本，关闭自动更新，解决问题。

4）赛博井字棋的多种解法

如果棋局正常，我们是不可能取得胜利，最多平局。
前端运行在用户本地，负责将数据展示和页面绘制，例如在本题中，将棋子数据显示在棋盘上就是前端的工作。而后端运行在服务器上，通常负责数据处理等幕后工作，例如本题里，井字棋 AI 就运行在后端。我们的落子位置会发送给后端，然后 AI 根据棋盘当前状态和我们的落子位置，计算出落子位置，并将落子后的棋盘状态返回给前端。
但是，出题人把检测一个格子能否落子的代码放在了前端。

解法1：“吃掉”AI 下在棋盘中央的这颗子。我们可以按 F12 键打开浏览器的开发人员工具，如下图，在弹出的界面中选择控制台，在 > 后输入指令 board[1][1]=0; setMove(1,1); 然后按 Enter 键执行指令。

解法2：直接删掉检测的代码setmove部分，发现棋盘任何地方都可以落子，包括已经有ai棋子的格子

async function setMove(x, y) {

解法3:非常有趣的是，这道题如果先后打开两个容器，其前端是不同步的，所以先下半盘，再在后一盘里往ai格子里落子，两容器前端同步之后你就把ai的棋子吃掉了。

4）acropalypse

修复裁剪过的图片,部分手机自带的系统截图工具存在漏洞，裁剪图片并不会删除原图，而是将裁剪后的图片的数据直接写在原图的开头，覆盖原先的数据，此时原图未被覆盖的部分仍然可以被恢复。一个修复此类图片的在线网站acropalypse

2）Airdrop

出题人提供了协议源码，通过Remix在线编译并执行合约用自己的钱包进行一次空投

// SPDX-License-Identifier: UNLICENSED
pragma solidity ^0.8.9;
contract SimpleContract {
 mapping(address=>uint)public balance;
 function airdrop() public {
  balance[msg.sender]=100;
 }
 function getbalance()public view returns(uint){
  return balance[msg.sender];
 }
}

等待Remix连接上自己的钱包，确认交易双方地址，执行空投操作
