WEEK 2 周报

本周GEEK坐牢，还有许多结课作业，没什么大进展。

张灏 2023/11/12

1. WEB

信息搜集

  • 服务器信息
  • 敏感目录
  • 敏感文件

通过http头中的Server等字段来确定后端是什么服务器，用了什么语言开发，是什么版本，初步确定漏洞范围。

工具：dirsearch（虽然之前新生赛的时候并没有派上用场）/ffuf

XSS

xss入门级

  • 大致了解富文本过滤

  • 学会了喜闻乐见的双写绕过

  • 注入点在标签属性中，引入新标签/插入新事件/标签的属性字段支持html编码

  • 存在单词黑名单可使用unicode编码绕过

中国剩余定理三素数RSA

之前写了一个程序，老是跑出抽象怪不说，耗时还长，这周把它拆成了两个程序，先求e，再解密，速度快了好几倍。

  • phi = (p-1)*(q-1)*(r-1)
e=1
while True:
    e=sympy.nextprime(e)
    try:
        d = gmpy2.invert(e,phi)
    except:
        continue
    else:
        if dP == d%((q-1)*(r-1)) and dQ == d%((p-1)*(r-1)) and dR == d%((p-1)*(q-1)):
            print(e)
            break

n=p*q*r
phi_n=(p-1)*(q-1)*(r-1)
